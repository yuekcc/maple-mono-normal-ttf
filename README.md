# Maple Mono 自用版

**本仓只保存构建产物，自用**，更多说明请参考 [Maple Mono](https://github.com/subframe7536/maple-font)。

基于 [Maple Mono V7.0 Beta23](https://github.com/subframe7536/maple-font/releases/tag/v7.0-beta23)，使用 `python build.py --normal` 构建。
